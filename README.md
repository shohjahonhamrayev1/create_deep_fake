  <h2 id="installation">Installation</h2>
  <ol>
    <li>Clone the repository:</li>
    <pre><code>git clone https://github.com/shohjahonhamrayev/create_deep_fake-repo.git</code></pre>
    <li>Install the required dependencies:</li>
    <pre><code>pip install -r requirements.txt</code></pre>
    <li>Download pre-trained models (if available) and place them in the <code>models/</code> directory.</li>
  </ol>

  <h2 id="usage">Usage</h2>
  <ol>
    <li>Prepare the source video and target face images:</li>
    <ul>
      <li>Source Video: The original video in which you want to replace the face.</li>
      <li>Target Face Images: A collection of images featuring the face you want to replace the source face with.</li>
    </ul>
    <li>Run the deepfake generation script:</li>
    <pre><code>python generate_deepfake.py --source_video path/to/source/video.mp4 --target_faces path/to/target/faces/ --output_video path/to/output/video.mp4</code></pre>
    <li>Wait for the script to process and generate the deepfake video.</li>
    <li>Find the generated deepfake video in the specified output directory.</li>
  </ol>
